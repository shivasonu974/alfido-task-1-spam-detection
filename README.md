# Task 1: Spam Detection using Machine Learning

## Objective
Build and evaluate a supervised classification model to classify SMS messages as spam or ham.

## Tools & Libraries
- Python 3.x
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab / Jupyter Notebook

## Environment Setup
```bash
pip install pandas numpy matplotlib scikit-learn
```

## Dataset
SMS Spam Collection Dataset
- Total messages: 5572
- Ham: 4825
- Spam: 747

## Steps Performed
1. Data loading
2. Data cleaning
3. TF-IDF feature extraction
4. Train-test split
5. Logistic Regression
6. Random Forest
7. Evaluation (Accuracy, Precision, Recall, F1, ROC-AUC)
8. Cross-validation

## Results
Best Model: Random Forest
- Accuracy: 97.67%
- Precision: 100%
- Recall: 82.67%
- F1 Score: 90.51%
- ROC-AUC: 98.68%
- Cross Validation Accuracy: 97.61%

## How to Run
Open the notebook in Jupyter or Google Colab and run all cells.
